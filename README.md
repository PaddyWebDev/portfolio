## Technologies Used

This portfolio is built using the following technologies:

- **Frontend**: 
  - React
  - Next.js
  - TypeScript
  - Tailwind CSS
  - ShadCN UI Components
  - Nodemailer (for the contact form email functionality)

- **Deployment**: 
  - Vercel (for hosting)
